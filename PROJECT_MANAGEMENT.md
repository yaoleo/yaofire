## 📊 项目概述

### 项目目标
创建一个全栈财富管理应用，帮助用户追踪收入、支出、资产和投资，提供财务分析和报表功能。

### 技术栈
| 层级 | 技术 | 版本 |

| **前端** | React | 19.2.0 |

## 📝 日常工作流程

### 每日工作步骤

1. **早上**: 检查 TODO 列表，选择今天的任务
2. **开发**: 编写代码，定期推送到 GitHub
3. **测试**: 本地测试，Railway 自动部署验证
4. **记录**: 更新 TODO 和进度
5. **总结**: 写 commit message 记录进度

### 版本控制规范

```bash
# 创建功能分支
git checkout -b feature/api-users

# 定期提交

# 推送到 GitHub（自动部署）
git push origin feature/api-users

# PR 合并到 main
# 自动部署到生产
```
